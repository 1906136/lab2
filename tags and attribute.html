DIFFERENCES BETWEEN HTML TAGS AND ATTRIBUTES

HTML TAGSHTML ATTRIBUTES

  1. Html tags hold html elements.

  2. Tags starts with < and ends with >.

  3. Tags are almost like keywords where every single tag has unique meaning.

  1. Html Attributes are used to describe the characteristics of an HTML element in detail.

  2. HTML Attributes are found only in the starting tag.

  3. HTML Attributes specify various additional properties to the existing html element
