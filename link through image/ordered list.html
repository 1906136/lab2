ORDERED LIST

Ordered(numbered) list are used to display a list of itemsthat should be in a specified order.

ORDERED LIST USES ONE SET OF< OL>< /OL>TAGS WRAPPED AROUND ONE OR MORE SETS OF < LI>< /LI> TAGS.


Ordered list can be displayed with several sequencing option.
The default in most browser is decimal number,but there are others available:
  • LETTERS
    • LOWERCASE ASCII LETTERS(a,b,c.......)
    • UPPERCASE ASCII LETTERS(A,B,C.......)
    • LOWERCASE CLASSICAL GREEK
  • NUMBERS
    • DECIMAL NUMBERS
    • DECIMAL NUMBERS WITH LEADING ZEROES(01,02,03.........)
    • LOWERCASE ROMAN NUMERALS(i,ii,iii,iv.............)
    • UPPERCASE ROMAN NUMERALS(I,II,III..............)
      • TRADITIONAL GEORGIAN NUMBERING(an,ban,gan......)
    • TRADITIONAL AMERICAN NUMBERING(mek,yerku,yerek.....)


    • EXAMPLE:   LIST OF INGREDIENTS USED IN MAKING TEA:
    1. SUGAR
    2. MILK
    3. WATER
    4. TEA LEAF
